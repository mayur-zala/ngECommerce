## Prerequisites

- Node.js (v16+)
- npm or yarn

## Installation

## Clone the repository

git clone <repository-url>

## Navigate to project

cd ngECommerce

## Install dependencies

npm install

## Start development server

npm start

Open `http://localhost:4200` in your browser.
